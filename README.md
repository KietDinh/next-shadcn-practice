## Project Setup

This project was initialized with `create-next-app` using the following configuration:

- **Project Name:** client
- **TypeScript:** Yes
- **ESLint:** Yes
- **Tailwind CSS:** Yes
- **`src/` Directory:** Yes
- **App Router:** Yes
- **Turbopack:** Yes
- **Import Alias:** Default (`@/*`)
